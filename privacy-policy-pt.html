📱 Nosso Compromisso com a Privacidade

Zero Coleta de Dados

Este aplicativo não coleta, transmite ou armazena quaisquer dados do usuário em servidores externos. Todos os dados são armazenados localmente no seu dispositivo.

Uso da Rede

Este aplicativo opera principalmente offline, mas possui funcionalidades limitadas que podem utilizar a conexão de rede:

  • Compras dentro do aplicativo: As transações da versão Pro se comunicam com os servidores da Apple (iOS) ou Google (Android) para processamento e verificação
  • Links em texto rico: Os links em notas de texto rico podem ser abertos no navegador do sistema (fora do aplicativo)
  • Reprodução de vídeo pela rede: Suporta reprodução opcional de vídeos de URLs de rede (apenas quando você escolher adicionar links de vídeo)
  • Compartilhamento do sistema: Ao usar a função de compartilhamento, o sistema operacional pode usar a rede para enviar conteúdo (e-mail, mensagens, etc.)

Observação: Todo o seu conteúdo pessoal (notas, fotos, vídeos, arquivos) permanece armazenado localmente e criptografado. O aplicativo não envia seu conteúdo pessoal pela internet.

🛡️ Armazenamento e Criptografia de Dados

Armazenamento Local

  • Todo o conteúdo de notas, anexos (fotos, vídeos, áudio, arquivos), tarefas, eventos de calendário e registros de check-in são armazenados localmente no seu dispositivo
  • Os dados são armazenados no diretório privado do aplicativo e não podem ser acessados por outros aplicativos
  • Não usamos nenhum servidor em nuvem para armazenar seus dados
  • O aplicativo suporta o modo multi-conta; os dados de cada conta são isolados e criptografados independentemente

Criptografia de Ponta a Ponta

  • Todos os dados do usuário são criptografados usando o algoritmo AES-256-GCM
  • As chaves de criptografia são derivadas da sua senha usando o algoritmo PBKDF2
  • As chaves de criptografia são armazenadas apenas na área de armazenamento seguro do dispositivo (iOS Keychain / Android Keystore)
  • As chaves de sessão são mantidas na memória apenas enquanto o aplicativo está desbloqueado e são automaticamente apagadas ao bloquear

Autenticação Biométrica

  • Você pode escolher usar impressão digital ou reconhecimento facial para desbloquear o aplicativo rapidamente
  • Os dados biométricos são gerenciados pelo sistema operacional; este aplicativo apenas recebe o resultado da verificação
  • Nunca acessamos, armazenamos ou transmitimos seus dados biométricos brutos

📝 Funcionalidades do Aplicativo e Processamento de Dados

Notas e Edição de Texto Rico

Você pode criar diários, notas Markdown, listas de tarefas e registros de gerenciamento de arquivos. Todo o conteúdo é armazenado e renderizado localmente apenas e nunca é enviado para um servidor.

Álbum e Gerenciamento de Mídia

O aplicativo suporta a adição de fotos, vídeos e arquivos de áudio às notas ou registros de check-in. Todos os arquivos de mídia são criptografados antes de serem salvos no armazenamento local e descriptografados em tempo real para visualização no aplicativo.

Check-in e Calendário

Os registros de check-in e os dados do calendário são gerados e armazenados completamente no local. A visualização do calendário lê apenas o horário de criação das notas locais e não envolve nenhum serviço de sincronização de calendário online.

Lixeira (Exclusão Suave)

As notas e dados excluídos são movidos primeiro para uma lixeira local, onde você pode restaurá-los ou excluí-los permanentemente. Os dados da lixeira também são armazenados com criptografia local, e não acessamos seu conteúdo excluído.

📤 Exportação, Importação e Compartilhamento

Exportação Criptografada e Backup

  • Você pode exportar seus dados como um arquivo de backup ENC criptografado
  • Os arquivos exportados são criptografados usando AES-256-GCM com a senha da sua conta atual
  • Os arquivos exportados são salvos no local escolhido no seu dispositivo (por exemplo, gerenciador de arquivos)

Importação

  • Você pode importar arquivos de backup criptografados locais ou arquivos de texto (.txt / .md) para o aplicativo
  • O processo de importação é realizado inteiramente no local; o conteúdo dos arquivos nunca é enviado para a rede

Compartilhamento

  • Quando você escolhe compartilhar uma nota ou anexo, o aplicativo invoca o painel de compartilhamento nativo do sistema operacional (como iOS Share Sheet / compartilhamento do sistema Android)
  • O comportamento de compartilhamento é tratado pelo sistema operacional e pelo aplicativo de terceiros que você seleciona; este aplicativo não acessa diretamente a internet para enviar dados

💎 Compras dentro do Aplicativo

A atualização para a versão Pro (compra única) é oferecida através dos sistemas nativos de compra dentro do aplicativo da Apple App Store (iOS) e Google Play (Android):

  • As transações de compra são processadas pela Apple ou Google; nós não coletamos nem armazenamos nenhuma informação de pagamento (como números de cartão de crédito ou endereços de cobrança)
  • Registramos apenas o status de desbloqueio (se a versão Pro foi comprada) no armazenamento seguro local para desbloquear as funcionalidades correspondentes neste dispositivo
  • A restauração de compras (Restore Purchases) se comunica diretamente com a Apple/Google. Nós não obtemos as informações da sua conta Apple ID ou Google

❌ Dados que Não Coletamos

  • ❌ Informações pessoalmente identificáveis (nome, e-mail, telefone, etc.)
  • ❌ Identificadores de dispositivo ou IDs de publicidade
  • ❌ Informações de localização
  • ❌ Estatísticas de uso ou dados de análise
  • ❌ Logs de falhas (a menos que você os envie ativamente através de canais do sistema)
  • ❌ Quaisquer dados transmitidos pela rede
  • ❌ Informações de pagamento (cartão de crédito, endereço de cobrança, etc.)

🌍 Direitos de Privacidade Regionais

Direitos dos Utilizadores da UE (GDPR)

  • Direito de ser informado - Compreender o processamento de dados através desta política
  • Direito de acesso - Aceder a todos os seus dados a qualquer momento
  • Direito de retificação - Editar e modificar os seus dados a qualquer momento
  • Direito de apagamento - Eliminar todos os dados desinstalando o aplicativo
  • Direito à portabilidade de dados - Obter uma cópia criptografada dos seus dados através da função de exportação

Direitos dos Utilizadores da Califórnia (CCPA)

  • Direito de saber - Não recolhemos qualquer informação pessoal
  • Direito de eliminação - Desinstale o aplicativo para eliminar os dados
  • Direito de opt-out - Não vendemos quaisquer dados
  • Direito de não discriminação - Todas as funcionalidades básicas estão disponíveis gratuitamente
