📱 Our Privacy Commitment

Zero Data Collection Principle

This application does not collect, transmit, or store any user data on external servers. All data is stored locally on your device.

Zero Network Connectivity

This application does not request Internet Permission and cannot access the Internet, fundamentally eliminating the possibility of data leakage through the network.

🛡️ Data Storage & Encryption

Local Storage

  • All note content, attachments (images, videos, files), to-do items and other data are stored locally on your device
  • Data is stored in the application's private directory and cannot be accessed by other applications
  • We do not use any cloud servers to store your data

End-to-End Encryption

  • All user data is encrypted using the AES-256-GCM encryption algorithm
  • Encryption keys are derived from your password using the PBKDF2 algorithm
  • Encryption keys are only stored in the device's secure storage area (iOS Keychain / Android Keystore)
  • Session keys are only retained in memory while the application is unlocked

Biometric Authentication

  • You can choose to use fingerprint or facial recognition to unlock the application quickly
  • Biometric data is managed by the operating system, and this application only receives verification results
  • We will never access, store, or transmit your original biometric data

📤 Export & Backup

Encrypted Export

  • You can export data as an encrypted backup file
  • Exported files are encrypted with the password you set
  • Exported files are stored in the location of your choice

❌ Data We Do Not Collect

  • ❌ Personal Identifiable Information (name, email, phone number, etc.)
  • ❌ Device identifiers
  • ❌ Location information
  • ❌ Usage statistics or analytics data
  • ❌ Crash logs (unless actively sent by you)
  • ❌ Any network transmission data

🌍 Regional Privacy Rights

EU User Rights (GDPR)

  • Right to Information - Understand data processing through this policy
  • Right to Access - Access all your data at any time
  • Right to Rectification - Edit and modify your data at any time
  • Right to Erasure - Delete all data by uninstalling the application
  • Right to Data Portability - Obtain a copy of your data through the export function

California User Rights (CCPA)

  • Right to Know - We do not collect any personal information
  • Right to Delete - Data can be deleted by uninstalling the application
  • Right to Opt-Out - We do not sell any data
  • Right to Non-Discrimination - All features are free of charge
